## How to use?

- download [nodejs](https://nodejs.org/en/download/)

- install dependencies with `npm`
```bash
npm install
```

- edit `src/config/app.ts`

- run start step
```bash
npm run start
```

**...or if you use `pm2` you can run it like this**
```bash
pm2 start "npm run start" -n FMDiscordStatus
```
